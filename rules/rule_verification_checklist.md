# 산출물 검증 프로토콜

Skill의 주요 작업 완료 후, 결과 출력 전에 자체 검증을 수행한다.

## 규칙

### 검증 항목
- 각 Step/subagent의 결과가 누락 없이 존재하는가
- 단계 간 산출물의 일관성이 유지되는가 (참조, 의존 관계 등)
- SKILL.md에 정의된 요구사항/기대 결과가 빠짐없이 반영되었는가

### 미충족 항목 처리
- 누락 또는 불일치 발생 시, 해당 Step 또는 subagent만 재실행한다.
- 재실행 후 검증을 다시 수행한다.
- 재실행으로 해결되지 않는 경우, Human에게 보고하고 진행 여부를 확인한다.

## 형식

### 검증 체크리스트
```
✅ 검증 체크리스트
- [x] 항목 1: 통과 사유
- [x] 항목 2: 통과 사유
- [ ] 항목 3: 미충족 사유 → 조치 계획
```
