# Step 0 변수 수집 프로토콜

Variables 섹션에 정의된 변수를 수집하는 Step 0에 공통 적용한다.

## 수집 규칙
- SKILL.md의 Variables 섹션에 "(필수)"로 표기된 변수는 반드시 확보한 후 다음 Step으로 진행한다.
- 필수 변수가 모두 확보될 때까지 다음 Step 진행을 금지한다.
- default 값이 명시된 변수는 Human이 별도 지정하지 않으면 default 값을 적용한다.
- "(선택)" 또는 "(optional)"로 표기된 변수는 미제공 시 스킵 가능하다.

## 확인 방식
- 수집된 변수 목록을 정리하여 Human에게 제시하고 확인받는다.
- Human이 일부 변수를 프롬프트에 이미 포함한 경우, 해당 값을 자동 추출하여 재질문하지 않는다.
