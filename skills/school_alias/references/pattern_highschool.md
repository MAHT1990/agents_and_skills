## 1. 유형 접미사 패턴

학교명 끝의 `[유형]고등학교`를 축약형으로 치환한다.
매칭 우선순위: **구체적 접미사 → 일반 접미사** (먼저 매칭된 1개만 적용).

### 1-1. 명시 패턴 (원본 제공)

| # | 원형 접미사 | 축약형 | 예시 (입력→맵핑) |
|---|-----------|--------|-----------------|
| 1 | 고등학교 | **고**, **고교** | 한국고 → 한국고등학교 |
| 2 | 여자고등학교 | **여고** | 한국여고 → 한국여자고등학교 |
| 3 | 남자고등학교 | **남고** | 한국남고 → 한국남자고등학교 |
| 4 | 과학고등학교 | **과고**, **과학고** | 한국과고 → 한국과학고등학교 |
| 5 | 공업고등학교 | **공고**, **공업고** | 한국공고 → 한국공업고등학교 |
| 6 | 상업고등학교 | **상고**, **상업고** | 한국상고 → 한국상업고등학교 |
| 7 | 사범대학부속고등학교 | **대부고**, **대부속고**, **사대부고**, **사대부속고** | 건국대학교대부고 → 건국대학교사범대학부속고등학교 |

| 9 | 대학교부속고등학교 | **대부고**, **대부속고** | 조선대학교대부고 → 조선대학교부속고등학교 |

### 1-3. 범용 축약 (고등학교→고/고교)

특수 유형 매칭 후에도, `고등학교` 부분은 항상 `고` / `고교`로도 축약 가능.
- 서울과학고등학교 → 서울과고 (유형축약) + 서울과학고 (범용) + 서울과학고교 (범용)

## 2. 대학교 접미 패턴

`XX대학교` + `사범대학부속/부설고등학교` 또는 `부속고등학교` 구조에서, 대학 이름을 축약.

### 2-1. 사범대학 부속/부설

**규칙**: 대학명 첫 음절 + 유형축약접미사
- 건국대학교사범대학부속고등학교 → **건**대부고, **건**사대부고, **건국**대부고, **건**대사대부고

| 원형 | 대표 축약형1 | 대표 축약형2 | 대표 축약형3 |
|------|---------|---------|----|
| 건국대학교사범대학부속고등학교 | 건대부고    | 건국대부고   |  건대사대부고 |
| 고려대학교사범대학부속고등학교 | 고대부고    | 고려대부고   | 고대사대부고 |
| 단국대학교사범대학부속고등학교 | 단대부고    | 단국대부고   | 단대사대부고  |
| 동국대학교사범대학부속고등학교 | 동대부고    | 동국대부고   | 동대사대부고  |
| 중앙대학교사범대학부속고등학교 | 중대부고    | 중앙대부고   | 중대사대부고  |
| 한양대학교사범대학부속고등학교 | 한대부고    | 한양대부고   | 한대사대부고  |
| 홍익대학교사범대학부속고등학교 | 홍대부고    | 홍익대부고   | 홍대사대부고  |
| 인하대학교사범대학부속고등학교 | 인대부고    | 인하대부고   | 인대사대부고  |

### 2-2. 대학교 직접 부속

**규칙**: 사범대학 경유 없이 `XX대학교부속고등학교` 구조. 유형 접미사 치환만 적용 (대학명 축약은 C 패턴에서 처리).

| 원형 | 축약형 |
|------|--------|
| 조선대학교부속고등학교 | 조선대부고, 조선대부속고 |
| 단국대학교부속소프트웨어고등학교 | 단국대학교부속소프트웨어고, 단국대학교부속소프트웨어고교 → 단대고 (C 패턴) |

## 3. 고유명사 축약 패턴

학교명에서 유형 접미사를 제거한 **고유명사 부분**이 3음절 이상일 때, 음절을 축약.

**규칙**:
- 4+ 음절: 2음절씩 끊어 각 첫 음절 추출 → `[0]+[2]+고`
- 3 음절: 첫 두 음절 추출 → `[0]+[1]+고`

| 원형 | 고유명사부 | 음절수 | 축약형 | 분할 |
|------|----------|--------|--------|------|
| 인천하늘고등학교 | 인천하늘 | 4 | 인하고 | 인천+하늘 → 인+하 |
| 민족사관고등학교 | 민족사관 | 4 | 민사고 | 민족+사관 → 민+사 |
| 인천포스코고등학교 | 인천포스코 | 5 | 인포고 | 인천+포스코 → 인+포 |
| 가재울고등학교 | 가재울 | 3 | 가재고 | 가재+울 → 가+재 |

## 4. 지역명 생략 패턴

학교명이 `지역명 + 고유명사부 + [유형]고등학교` 구조일 때, 지역명을 제거하여 축약형을 생성.

**규칙**: 지역명 접두사를 제거한 `고유명사부 + 축약접미사`를 기존 축약형 **앞에** 배치.
130+개 광역시·시·군 지역명을 긴 이름부터 매칭 (남양주→남, 서귀포→서 방지).

**제외 조건**:
- 고유명사부 전체가 지역명인 경우 제외 (인천고등학교 → ~~고~~)
- `대학교` 접미 패턴 제외 (부산대학교사범대학부설고등학교 → 대학교 축약만 적용)

| 원형 | 새 축약형 (앞에 추가) | 기존 축약형 (유지) |
|------|---------------------|-------------------|
| 고양동산고등학교 | **동산고**, **동산고교** | 고양동산고, 고양동산고교, 고동고 |
| 대전가오고등학교 | **가오고**, **가오고교** | 대전가오고, 대전가오고교, 대가고 |
| 대전대성여자고등학교 | **대성여고**, **대성여자고**, **대성여자고교** | 대전대성여고, 대전대성여자고, ... |
| 천안쌍용고등학교 | **쌍용고**, **쌍용고교** | 천안쌍용고, 천안쌍용고교, 천쌍고 |
| 인천고등학교 | (없음 — 지역명=전체) | 인천고, 인천고교 |
| 서울과학고등학교 | (없음 — 지역명=전체) | 서울과고, 서울과학고, 서울과학고교 |

## 5. Edge Case

### 5-1. Type 1: 축약형이 실재 학교명과 충돌 — 0건 (현재)

축약형을 원형으로 복원시 **다른 학교명**이 되는 경우.
현재 데이터셋(2,294교)에서는 미발생. 신설 학교 시 발생 가능.

### 5-2. Type 2: 동일 축약형 → 복수 학교 (1:N 모호성) — 168건

| 원인 | 건수 | 대표 사례 |
|------|------|----------|
| 고유명사축약 충돌 | 128 | 경기고 → 경기고등학교 / 경기기계공업고등학교 등 6교 |
| 유형축약 충돌 | 22 | 경기과고 → 경기과학고등학교 / 경기북과학고등학교 |
| 성별축약 충돌 | 10 | 부여고 → 부산여자상업고등학교 / 부여고등학교 |
| 대학부속 충돌 | 8 | 전대부고 → 전남대 / 전북대 / 전주대 |

### 5-3. 주의: 잘못된 역확장 방지

축약형 `X고`에서 역확장 시, 접미사를 **잘못 분리**하면 안 된다.

| 입력 | 잘못된 역확장 (X) | 올바른 역확장 (O) | 원인 |
|------|-----------------|-----------------|------|
| 부여고 | 부+여자고등학교 | 부여+고등학교 | `여`가 고유명사의 일부 |
| 경남고 | 경+남자고등학교 | 경남+고등학교 | `남`이 고유명사의 일부 |
| 사과고 | 사+과학고등학교 | 사과+고등학교 | `과`가 고유명사의 일부 |
| 사공고 | 사+공업고등학교 | 사공+고등학교 | `공`이 고유명사의 일부 |
| 경상고 | 경+상업고등학교 | 경상+고등학교 | `상`이 고유명사의 일부 |
